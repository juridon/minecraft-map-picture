マイクラの地図にドット絵描くプログラムを書くやつ

教育版のMakeCode用です。

コードビルダーでMakeCodeを選んでプロジェクトを新規作成
jsを選んで全部削除→ココで作成したコードを貼り付けてね

画像はサイズ128×128がおすすめです。

再生ボタンを押したら、コマンド「run」で
run num1 num2 num3 のように使います。座標(num1, num2, num3)から128×128の範囲がドット絵になります。
透過部分は何もかかれないので先にキャンバスになる床を準備しておくことをおすすめします。
地図にしたい場合は初期位置の座標(num1, num2, num3)を調整してね。
x,z座標は128の倍数-64の値がいいよ。y座標はスーパーフラットなら-60がいいよ。
例：run -64 -60 -192
コマンド叩いた直後は動かないほうがいいかな。

★注意事項★

まずはテスト用にワールドを作成して、十分試してから使って下さい。

実行後、ドット絵を描き終わるまでPCのスペックによるかもですがwin10版で30分～40分程度かかりました。

このアプリで作成したコードを使用してどんな結果になっても、自己責任となります。

公式とは一切関係ありません。
